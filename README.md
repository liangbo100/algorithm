## 算法

### 什么是算法

- 有具体的问题
- 有设计解决这个问题的具体流程
- 有评价处理流程的可量化指标


### 算法的分类

- 分类当然非常多
- 对于新手学习特别重要的一个分类：
  - 明确知道怎么算的流程
  - 明确知道怎么试的流程

### 数据结构

大致分为两种： 连续结构（线性）， 跳转结构（非线性）

- 数据结构是存储、组织数据的方式
- 精心选择的数据结构可以带来更高的运行或者存储效率
- 数据结构是很多算法得以进行的载体

### 最基本的数据结构

- 数组
  - 便于寻址，不便于增删数据
- 链表
  - 便于增删数据，不便于寻址
